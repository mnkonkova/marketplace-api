copyleft (GPL / LGPL / AGPL / MPL / SSPL) dependencies are present.
`go-licenses check ./... --disallowed_types=forbidden,restricted` passes.

## How to regenerate

```bash
go install github.com/google/go-licenses@latest
go mod download
go-licenses report ./... 2>/dev/null > /tmp/licenses.csv
go-licenses check ./... --disallowed_types=forbidden,restricted --ignore=marketpclce
```

The first command produces a `package,license_url,license_type` CSV; group by
the third column to produce this report. The check command exits non-zero if a
copyleft license sneaks in via a new dependency.
